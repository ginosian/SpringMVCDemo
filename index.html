Project description:

There are 2 embedded authorization levels - “ADMIN” and “USER”. Users with role “ADMIN” are allowed to create users, create projects, create tasks, assign tasks to users and projects, reassign tasks, and modify projects and tasks. Users with role “USER” are allowed to view tasks assigned to him/her and reassign tasks to other “USER”-s.

Project technology:

  • Core framework – Spring (Including Spring’s Authentication, Authorization and “Remember me” functionality).
  • Testing and documentation TDD approach. JUnit, Mockito, Hamcrest
  • Configurations - Annotation based.
  • DB-ORM – MySQL, Hibernate ORM
  • Deployment - Application server is Apache Tomcat 7 and it is deployed on Openshift RHCloud
